# 🚀 NLP Preprocessing Application using Streamlit

## 📌 Project Description

This project is an interactive Natural Language Processing (NLP) application developed using Python and Streamlit. The application demonstrates various fundamental NLP techniques used in text analytics and machine learning.

Users can enter any text and explore different NLP preprocessing methods such as Tokenization, Text Cleaning, Regular Expression Processing, Stemming, Lemmatization, Bag of Words (BoW), TF-IDF, and Word Embeddings through a simple web interface.

The project is designed for educational and practical learning purposes to help understand how raw text is transformed into machine-readable data.

---

## 🎯 Project Objectives

* Perform essential Natural Language Processing tasks.
* Demonstrate text preprocessing techniques interactively.
* Convert unstructured text into structured representations.
* Visualize NLP outputs in a user-friendly manner.
* Provide hands-on experience with popular NLP libraries and tools.

### 1. Text Processing Module

### 2. NLP Analysis Module

# 🧠 Text Processing Module

The application preprocesses user-entered text using multiple NLP techniques.

### Preprocessing Tasks

* Sentence Tokenization
* Word Tokenization
* Character Tokenization
* Text Cleaning
* Lowercase Conversion
* Stop Word Removal
* Punctuation Removal
* Number Removal

The processed text becomes more suitable for machine learning and text analysis tasks.

---

# 🔍 NLP Analysis Module

The application provides several NLP techniques for analyzing text.

### Tokenization

Breaks text into:

* Sentences
* Words
* Characters

### Text Cleaning

Removes:

* Special characters
* Punctuation
* Numbers
* Stop words

### Regular Expressions

Extracts:

* Email addresses
* Numerical values

### Stemming

Implements:

* Porter Stemmer
* Lancaster Stemmer

### Lemmatization

Uses SpaCy to:

* Identify Parts of Speech (POS)
* Generate root word forms (lemmas)

### Bag of Words (BoW)

* Converts text into word-frequency representations.
* Displays term occurrence counts.

### TF-IDF

* Calculates Term Frequency–Inverse Document Frequency scores.
* Highlights important words within the text.

### Word Embeddings

* Uses SpaCy word vectors.
* Represents words numerically for machine learning applications.

---

# 🌐 Streamlit Web Application

Streamlit is used to create an interactive and responsive web interface.

### Features

* Text input area for custom text
* Sidebar menu for selecting NLP techniques
* Real-time text processing
* Interactive tables for displaying results
* User-friendly visualization of NLP outputs

### Workflow

1. User enters text.
2. Selects an NLP technique.
3. Clicks the Process Text button.
4. Application performs the selected operation.
5. Results are displayed instantly.

# ⚙️ Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/RAISA2005/TRAININGPROGRAMPROJECTS.git
```

### Navigate to the Project Directory

```bash
cd TRAININGPROGRAMPROJECTS
```

### Install Required Dependencies

```bash
pip install streamlit nltk spacy pandas scikit-learn matplotlib
```

### Download NLTK Resources

```python
import nltk
nltk.download('punkt')
nltk.download('stopwords')
```

### Install SpaCy Language Model

```bash
python -m spacy download en_core_web_sm
```

### Run the Application

```bash
streamlit run project1.py
```
